## Getting started

**Installing**
> Cloning the repository

```bash
$ git clone https://github.com/biagavirete/calculator.git
$ cd calculator
```

**Running**
> Installing dependencies

```bash
$ yarn
```

> Running React

```bash
$ yarn start
```
